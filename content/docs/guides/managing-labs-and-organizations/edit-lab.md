# Edit a lab

> [!WARNING]
**The Permissions required for this operation are Admin**
Lab names and descriptions can be edited through the UI:
1. Click **Labs** → select the lab
1. Click **Edit**
1. Update the description or other editable fields
1. Click **Save**

The lab's Google Project ID (used for GCP resource naming) cannot be changed after creation.
